# 🌲 Optimisation du seuil de coupe d'une forêt

## Description

Ce projet explore l'application des mathématiques et de l'informatique à une problématique concrète de gestion forestière.
L'objectif est de **modéliser la croissance d'une forêt** puis d'**optimiser le seuil de coupe** afin de maximiser le rendement en bois sur le long terme.

## Objectifs

- Construire un modèle mathématique de croissance forestière
- Implémenter ce modèle en Python
- Déterminer le seuil de coupe optimal à l'aide de méthodes d'optimisation

## Outils utilisés

- **Python** — modélisation et optimisation
- **Jupyter Notebook** — mise en page interactive des résultats
- **LaTeX** — rédaction du rapport final
